# Diamond E-shop

A basic mock diamond e-shop made for practice with PHP.

## Functionality:

- Simple user system:
    - Admin & regular users:
        1. admin
        2. fares (regular)
        3. john (regular)
        4. jane (regular)
    - User login (Password same as username)
- Simple order system:
    - Adding items to the cart and sending orders
    - Admin can complete orders
